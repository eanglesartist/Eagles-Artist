# 🎬 AI Cinematic Studio

Professional AI video generation workspace built with Streamlit.

## Features

- 🎥 AI Movie Generator
- 🎵 Music Video Generator
- 🎬 Short Film Generator
- ✨ Video Extend
- 🖼 Image Reference
- 📖 Storyboard
- 🎞 Timeline Editor
- 📦 Export MP4

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
